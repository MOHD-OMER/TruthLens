Project Overview

TruthLens —
AI-Powered Fake News Detection

A deep learning system that classifies news articles and headlines using a hybrid CNN-LSTM architecture trained on labeled real and fake news corpora. Built as an end-to-end production Flask application with database logging, REST API, and real-time analytics.

Model Architecture

🔤
Embedding Layer

Learned word embeddings map vocabulary tokens into dense semantic vectors for downstream processing.

🧱
Conv1D (CNN)

Convolutional filters extract local n-gram patterns and keyword-level features from text sequences.

🔄
LSTM Layer

Long Short-Term Memory captures long-range sequential dependencies and discourse-level signals.

🎛
Dropout

Regularization layers prevent overfitting on training data and improve generalization to unseen text.

⚡
Dense + Sigmoid

Fully connected output layer produces a probability score in [0, 1]: closer to 1 signals real news.

📐
Sequence Length

Text is tokenized and padded to a fixed 100-token window for consistent model input shape.
